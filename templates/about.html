THE ORDER AND MANAGEMENT SYSTEM FOR DENTAL BUSINESS

I have created an e-commerce web application for dental business order and management system. In this application, the user has three options clinic, lab, manufacturer while registering. The manufacturer can add and delete products which can lab user able to see when he/she logged in. Lab user can either add and delete products, or it can buy it from a manufacturer. A client can order products from a lab.
